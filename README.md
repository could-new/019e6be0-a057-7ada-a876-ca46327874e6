# Buscador de Precios - Pickguards

Una aplicación en Flutter diseñada para ayudar a los músicos en Argentina a encontrar el mejor precio para repuestos específicos de instrumentos, con un enfoque actual en **Pickguards Nacarados (Pearloid) para Fender Precision Bass**.

## Características

- **Comparativa de Precios**: Muestra el mejor precio disponible y alternativas.
- **Foco en Repuestos Específicos**: Actualmente configurado para mostrar el mejor precio de **Pickguards Nacarados para Fender Precision Bass** en Argentina.
- **Simulación de Tiendas**: Visualiza opciones de compra en tiendas reconocidas o plataformas como MercadoLibre.
- **Diseño Responsivo**: Funciona y se ve increíble en dispositivos móviles, tablets y web.

## Tecnologías Utilizadas

- **Flutter / Dart**: Para el desarrollo de la interfaz de usuario multiplataforma.
- **Material 3**: Para un diseño moderno y atractivo.

## Cómo ejecutar el proyecto

1. Asegúrate de tener [Flutter](https://flutter.dev/docs/get-started/install) instalado en tu sistema.
2. Clona este repositorio o descarga el código fuente.
3. Abre una terminal en la carpeta del proyecto.
4. Ejecuta `flutter pub get` para obtener las dependencias (si aplica en el futuro, actualmente no hay externas).
5. Ejecuta `flutter run` para probar la aplicación en tu dispositivo o emulador.
