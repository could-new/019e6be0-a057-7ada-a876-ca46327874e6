# Buscador de Precios: Pickguard Torstein

Esta es una aplicación generada para comparar y encontrar el mejor precio de un **Pickguard Torstein para Fender Precision Bass** en Argentina. 

## Características
- **Interfaz Moderna**: Diseño centrado en el producto con esquema de colores estilo "Sunburst".
- **Destacado de Oferta**: Tarjeta principal que resalta el precio más bajo encontrado simulando una tienda real (ej. MercadoLibre).
- **Lista de Comparación**: Muestra alternativas de precios en otras tiendas de instrumentos musicales locales (ej. Fusión Musical, Casa Libertella, etc.).
- **Diseño Responsivo**: La aplicación se adapta fluidamente a pantallas de dispositivos móviles, tablets y escritorios, centrando el contenido de forma óptima.

## Tecnologías Utilizadas
- **Flutter**: Framework principal de la aplicación.
- **Material Design 3**: Para componentes UI modernos, tarjetas enriquecidas y tipografía atractiva.

## Ejecución

Para correr esta aplicación localmente:
```bash
flutter run
```
